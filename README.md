## Development

```sh
git clone https://github.com/paul-c-hartman/calsync
cd calsync
bundle install
bin/rails db:prepare
```

To start the server:

```sh
bin/dev
# If it errors, you may need to:
gem install foreman
```

Running linter and tests:

```sh
# Linting
bin/rubocop
# Tests
bin/rails spec
```
